-- Gestión de videojuegos --

Descripción programa:
=> Programa gestor de datos sobre una colección de videojuegos.
=> El usuario puede registrar datos sobre cada uno de los videojuegos de su colección como por ejemplo 
(Título, año de publicación, empresa, plataforma, imagen de portada...)

Interfaz Usuario:
